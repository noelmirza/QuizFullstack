## Introduction

This is a full-stack quiz application that allows users to take quizzes and see their results. The front-end is built with React.js and Material UI, while the back-end is powered by an ASP.NET Core API.

## Features

- User authentication
- Multiple choice questions
- Timed quizzes
- Detailed results and scoring
- Responsive design using Material UI

## Installation

### Prerequisites

- [.NET 6 SDK](https://dotnet.microsoft.com/download/dotnet/6.0)
